Methodology

Suppose there exists some feature set \(x\) such that its linear transformation maps to the ground-truth (object/crowd) density function i.e. \[ F_i(p) = w^T x_p^i \] whereas \(x_p^i\) is feature value at pixel \(p\) in image \(i\). Then this linear transformation can be learned by minimizing the following equation \[ w = \underset{w}{\arg\min} \bigg( w^T w + \lambda \sum_{i=1}^{N} dist\big( F_i^0(.), F_i(. \vert w)\big ) \bigg) \]

Given a set of images \(I_1, I_2, ..., I_N\) and a ground-truth density map \(F_i^0(p)\) for each image \(i\) such that \(\sum_{p \in I_i} F_i^0(p)\) equals the total number of objects present in the image then a set of parameters \(w\) can be learned such that the mapping function \(F_i(p \vert w)\) results in minimum error over the whole dataset.
